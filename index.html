Qur’an manners

  • Keep the Qur’an clean and high.
  • Listen quietly.
  • Recite with care.
Surah

  • Learn slowly.
  • Repeat small parts.
  • Understand that we ask Allah for guidance.
Listening

Listen and repeat

  • Listen first.
  • Repeat gently.

👨‍👩‍👧 Parent Corner

Helpful notes for parents and teachers using the homepage with young children.

How to use

Keep lessons short. Let the child choose one picture card, complete one activity, and celebrate effort with praise.

Best routine: Topic → Question → Practice → Game → Star.

Safe learning

Use the website with gentle adult guidance. For Arabic recitation and pronunciation, parent or teacher correction is important.
